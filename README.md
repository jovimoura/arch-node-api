# Arch Node API

<p>
  API Node Architecture with Unit Test's.
</p>

## Installation

### Clone

```
git clone 
```

### Access the folder

```
cd <name-folder>
```

### Install dependencies

```
npm i
```

### Start test's

```
npm test
```
